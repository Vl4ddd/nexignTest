# nexignTest

## Инструкция для запуска  
git clone https://github.com/Vl4ddd/nexignTest  
cd .\nexignTest\service  
mvn clean install    
mvn spring-boot:run 

Сервис запускается на localhost:8080

## Endpoints 
Post /api/v1/cdr/generate  
Get /api/v1/udr/all  
Get /api/v1/udr/subscriber
